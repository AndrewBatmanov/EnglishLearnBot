# EnglishLearnBot (by Andrew Batmanov - pd_fpy-134)
***
## Телеграм-бот для изучения английского языка
***

## Структура проекта:
### main.py 
Точка входа для запуска бота
### bot.py
Основная логика бота
### database.py
Работа с базой данных (PostgreSQL)
### parameters.py 
Конфигурация и настройки (включая токен бота и пароль от базы данных), а также первые 10 слов
### .gitignore 
Игнорируемые файлы (ничего важного)
***
## Запуск бота
### 1. Сделайте форк этого репозитория
### 2. Откройте папку в вашей программе-интерпретаторе 
(Я использовал PyCharm)
### 3. Создайте своего бота
(Для этого найдите в телеграме @BotFather)
### 4. Скопируйте токен бота и вставьте его в parameters.txt
### 5. Установите необходимые расширения
(Открыв терминал, введите команду pip install -r requirements.txt)
### 6. Вставьте пароль от Вашей базы данных в parameters.py
### 7. Запустите бот через main.py
### 8. Перейдите в созданный Вами бот в телеграм и введите /start
